# Code of Conduct — Not X Rippers Esports Website

## Our Commitment
We are committed to providing a welcoming, inclusive, and harassment-free
environment for everyone, regardless of age, gender, experience, nationality,
or background.

## Our Standards
Positive behavior includes:
• Respectful communication  
• Constructive feedback  
• Supporting new contributors  
• Using inclusive language  

Unacceptable behavior includes:
• Harassment or discrimination  
• Abusive language  
• Personal attacks  
• Spam or malicious contributions  

## Responsibilities
Project maintainers are responsible for:
• Reviewing contributions fairly  
• Maintaining a positive community  
• Enforcing community standards  

## Scope
This Code of Conduct applies to:
• GitHub repository  
• Project discussions  
• Community channels  
• Events and collaborations  

## Enforcement
Violations may result in:
• Warning  
• Temporary restriction  
• Permanent ban
